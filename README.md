# Ironwood Landscaping

Desert-smart landscaping for Tucson, AZ. Design, installation, irrigation, maintenance, tree care, and seasonal cleanups.

**Live:** https://ironwood-landscaping-three.vercel.app

## Tech Stack

- Next.js 14 (App Router)
- Tailwind CSS
- Supabase (leads DB)
- Resend (email notifications)

## Getting Started

```bash
cp .env.example .env.local
npm install
npm run dev
```
